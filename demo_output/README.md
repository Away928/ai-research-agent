# Demo 输出目录

本目录由 AI Research Agent 运行时自动生成，包含：

| 文件 | 说明 |
|------|------|
| `*_行业扫描报告_*.md` | 4 阶段流水线自动生成的行业研究报告 |
| `_history.json` | 历史研究记录索引（最近 50 条） |

## 生成流程

1. **数据采集** — 新浪/腾讯/baostock/巨潮/AKShare 多源并发，单源失败不阻塞
2. **行业全景** — AI 分析行业定义、市场空间、产业链、政策技术
3. **公司深度** — 头部公司对比（ROE/毛利率/增速/负债率/PE）
4. **投资研判** — 周期判断、Red Team 反方观点、风险矩阵 → 最终报告

每份报告含 AI 质量校验日志（数字引用密度、来源标注、不确定性标注、结构完整性）。

## 重要声明

- Demo 报告中的投资建议仅供展示研究框架，**不构成投资建议**
- 报告由 AI 辅助生成，数据来源标注于报告正文中
- 无 API key 时系统自动降级为 Prompt 模板输出模式，仍可手动使用
