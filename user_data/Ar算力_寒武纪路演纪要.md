# 寒武纪2026年中期业绩路演纪要
## 2026-07-05
来源：寒武纪IR部门

管理层表示，2026年Q2公司AI训练芯片出货量同比增长超过200%，主要受益于国内大模型训练需求的持续爆发。目前7nm工艺的思元590芯片良率已稳定在85%以上，单卡算力达到512TFLOPS（FP16），性能接近英伟达H100的70%。

公司计划在Q3推出新一代思元690芯片，采用更先进的Chiplet架构，预计性能将有50%以上的提升。毛利率方面，随着规模化效应释放，管理层预计全年毛利率有望达到60%以上，高于此前55%的指引。

竞争格局方面，公司认为华为昇腾是最大竞争对手，但在独立第三方芯片供应商的定位上有差异化优势。特别是在互联网客户中，中立性是一个重要卖点。

风险方面，管理层坦承出口管制仍是最大的不确定性，公司已在积极布局国产供应链替代方案。
